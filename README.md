# AI Thumbnail Generator

- MERN Stack Project
- Uses Gemini API + Cloudinary
- Generates YouTube-style thumbnails

## Setup

cd server
npm install
npm run dev

cd Client
npm install
npm run dev

## Notes

- The `Client/src/assets/` folder is missing the original `hero_img.png` and
  `thumb_1.jpg` ... `thumb_7.jpg` files. The code currently uses public
  placeholder URLs as a fallback so the project builds and runs. To restore
  bundled assets, drop the files in `Client/src/assets/` and switch the
  imports back to static `import` statements (see comments in
  `Client/src/components/tilt-image.tsx` and `Client/src/assets/assets.ts`).
- `server/dist/` is generated by `npm run build`; it is not committed.
